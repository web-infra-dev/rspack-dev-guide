Rspack Development Guide

Tuesday meeting

Process

  1. Go through unsolved issues/PRs in the milestone that plan to finish last week. Link

    1. Move unsolved issues/PRs in the milestone that plan to finish this week.
    2. Close the milestone that plan to finish last week.

  2. Go through issues in planned milestone.

    1. Prioritize all all issues. Link

    2. Go though p1 issues. Link

    3. Go though p2 issues. Link

    4. Go though p3 issues. Link

    5. Assign issues.

      1. Assign issue to related persons.

      2. Assign issue to the milestone that plan to finish.

  3. Review issues/PRs in the milestone plan to finish this week.

Discussions

  1. Go through issues/PRs with the label named to be discuss. Link

  2. Go through discussions with the label named to be discuss. Link
